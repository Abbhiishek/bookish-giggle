To set those we simply set git configs like username and email for the user. To do so follow these commands : ```bash $ git config --global user.name "YOUR_USER_NAME" ``` ```bash $ git config --global user.email "YOUR_EMAIL" ``` After setting up you can check if they are configured well or not through these commands : ```bash git -config --global user.name git -config --global user.email ``` These would return your given values or string. ![git configuration](https://dev-to-uploads.s3.amazonaws.com/uploads/articles/hft1qj0y631cix7n2atp.png) Now we are actually ready for the project so head over and create a folder and add any project you want to push over to github and keep a track of changes in git. ![git init in folder](https://dev-to-uploads.s3.amazonaws.com/uploads/articles/nrjd5p8z5j0uyrbt352q.png) As we are done with creating a git repository , its time to add some files and commit the changes to log.üòé For this article am using a simple fronted site .
